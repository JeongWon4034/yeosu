# 🌊 여수 해양 쓰레기 모니터링 & 예측 대시보드

이 프로젝트는 여수 지역의 섬별 쓰레기 유입량을 예측하고 수거 경로를 최적화하는 시스템입니다.

## 📊 대시보드 데이터 구성 현황

대시보드의 완성도 높은 시연을 위해 **실제 분석 데이터**와 **시뮬레이션 데이터**를 병행하여 활용하였습니다.

### 1. 실측 및 실제 분석 데이터 (Real Data)
이 데이터들은 실제 관측 및 모델링(MOHID, LSTM, GNN)을 통해 도출된 핵심 결과물입니다.

| 구분 | 상세 내용 | 관련 파일 |
| :--- | :--- | :--- |
| **섬 위치 (356개)** | 국가기본도 기반 여수 지역 섬 목록 및 좌표 | `yeosu_islands_wgs84.csv` |
| **섬 속성 데이터** | 유무인도 여부, 연륙교 여부 등 행정 데이터 | `yeosu_islands_wgs84.csv` |
| **오염원 방출점** | 해수욕장, 어업, 항구 등 실제 위치 (127개 지점) | `release_points_encoded.csv` |
| **해양 시계열** | KHOA+기상청 실측 (2025.01 ~ 2026.04) | `env_timeseries_merged.csv` |
| **실시간 기상** | KHOA 공공 API 실시간 호출 데이터 | API 연결 |
| **LSTM 모델 성능** | 23개 관측소 교차검증 (R²=0.917, MAE=516) | `loocv_results.csv` |
| **항로 탐색** | 실제 해안선 지형 기반 A* 알고리즘 경로 | GeoPandas 지형 데이터 |

### 2. 시뮬레이션 데이터 (Simulation Data)
보안 및 데이터 희소성(MOHID 데이터 기간 한계 등)으로 인해 기능 시연을 위해 가공된 데이터입니다.

| 항목 | 가공 이유 및 방식 |
| :--- | :--- |
| **섬별 쓰레기 유입량** | MOHID 데이터 기간 한계로 인한 3계층 확률 난수 생성 (저/중/고위험군) |
| **집중투입 지수 (Risk)** | 가상 `trash_cnt`를 기반으로 정규화하여 산출 |
| **쓰레기 유형** | 실측 유형 데이터 부재로 5가지 유형 무작위 조합 |
| **향후 예측 그래프** | 모델의 대응력 시연을 위해 현재 Risk 기준 랜덤 워크 추정 |
